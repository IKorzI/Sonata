# AFK Minecraft Launcher

Лаунчер Minecraft, построенный с использованием Electron и Svelte.

## Установка

```bash
npm install
```

## Разработка

Для запуска в режиме разработки:

```bash
npm run dev
```

Это запустит Vite сервер разработки на http://localhost:3000

## Сборка

Для создания production сборки:

```bash
npm run build
```

## Запуск Electron

Для запуска Electron приложения в режиме разработки:

```bash
npm run electron:dev
```

Для создания production сборки Electron приложения:

```bash
npm run electron:build
```
